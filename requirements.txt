# INSTALLATION NOTES

# 1. Install PyTorch first with CUDA support:
#    pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121
#
# 2. Install this requirements file:
#    pip install -r requirements.txt
#
# 3. Install PyTorch3D:
#    python ./SyncTalk-FastAPI/scripts/install_pytorch3d.py
#
# 4. Install SyncTalk custom CUDA modules:
#    cd SyncTalk-FastAPI
#    pip install ./freqencoder
#    pip install ./shencoder
#    pip install ./gridencoder
#    pip install ./raymarching
#
# 5. Set up environment variables in rapido/.env:
#    ELEVEN_API_KEY=your_api_key
#    SYNCTALK_SERVER_URL=http://localhost:8000
#
# 6. Download SyncTalk model files:
#    - May.zip -> SyncTalk-FastAPI/data/May/
